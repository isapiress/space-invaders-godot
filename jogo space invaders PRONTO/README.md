\# Space Invaders



Projeto desenvolvido no primeiro ano do curso Técnico em Informática, durante a disciplina de Desenvolvimento de Jogos.



O objetivo do projeto foi recriar uma versão simples do jogo Space Invaders utilizando a Godot Engine, com foco no aprendizado de lógica de programação, movimentação de personagens, colisões e estruturação de jogos 2D.



\## Tecnologias utilizadas

\- Godot Engine

\- Python
